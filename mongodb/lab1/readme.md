* El dataset del trabajo de clase es: 

Sample - Sample - Superstore.csv

* Utiliza los datos:

countries.json

* Con base en ambos archivos, cada equipo/estudiante debe crear un documento json llamado ‘superstore.json’ que contenga algunos registros de datos a partir de los datos integrados de ambos archivos.

* Este documento ‘superstore.json’ será almacenado en una colección mongodb bajo una base de datos a crear (ej: si3009db) -> colección: ‘superstore’

## Primer Lab – Base de datos documentalesMondoDB

* Realizar la instalación de MongoDB en tu máquina.

* Crear una cuenta en Mongo Atlas, crear un cluster y usuario para conexión remota.

* Realizar la instalación de MongoDB Compass en tu máquina local, para conectar con Mongo Atlas y la BD local de MongoDB.

* Cree una base de datos tanto local como en MongoAtlas, llamada: ‘si3009db’

* Crear una colección: ‘superstore’, en el cual cargará los datos de: ‘superstore.json’
